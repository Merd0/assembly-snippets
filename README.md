# MIPS32 Assembly Master Repository

This repository provides a bilingual (Turkish/English) MIPS32 Assembly learning path.

- `TR/` contains Turkish instructional material.
- `EN/` contains English instructional material.
- Folder structures and filenames are symmetric across both languages.
