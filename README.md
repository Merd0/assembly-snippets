# MIPS32 Assembly Master Repository

Bilingual (TR/EN) MIPS32 Assembly curriculum designed for progressive, hands-on mastery from fundamentals to system-level topics.

- `TR/` -> Turkish learning track
- `EN/` -> English learning track
- Folder structure and filenames are symmetric across both tracks

## Curriculum Status (Current)

- ✅ Stage 00: Study Roadmap
- ✅ Stage 01: Hello MIPS
- ✅ Stage 02: Arithmetic and Logic
- ✅ Stage 03: Control Flow
- ✅ Stage 04: Memory and Arrays
- ✅ Stage 05: Functions and Stack
- ✅ Stage 06: Floating-Point FPU
- ✅ Stage 07: Macros
- ✅ Stage 08: Interrupts and Exceptions
- ⏳ Stage 09: Mini Projects (next)
