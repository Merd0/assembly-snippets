# 02 - Weekly Plan (Simple, Sustainable, Serious)

## Recommended pace

- **5 sessions/week**
- **60-90 min/session**
- **One rest/review day every 6th day**

## Session template (use every time)

1. **Warm-up (10 min):** read previous notes + rerun yesterday's example.
2. **Core study (30-45 min):** understand one new file line-by-line.
3. **Hands-on change (15-20 min):** modify the file and verify behavior.
4. **Reflection (5-10 min):** write what broke, what fixed it, what you learned.

## 8-week default roadmap

- **Week 1:** Stage 01 + start Stage 02
- **Week 2:** Finish Stage 02 + Stage 03
- **Week 3:** Stage 04
- **Week 4:** Stage 05
- **Week 5:** Stage 06
- **Week 6:** Stage 07 + Stage 08 foundations
- **Week 7:** Stage 08 deep practice (exception/MMIO)
- **Week 8:** Stage 09 mini-project implementation + cleanup

## If you are busy (fallback mode)

If life gets busy, switch to:

- 3 sessions/week
- 45-60 min/session
- Keep the same stage order (just slower)

Consistency beats intensity.
